Через GitHub Issue

Простой способ

Создайте Issue с описанием вашего конфига. Мы сами добавим его в базу.

  • Заполните форму Issue
  • Приложите код конфига
  • Опишите особенности
  • Ждите добавления
Создать Issue

Через Pull Request

Для продвинутых

Сделайте форк репозитория, добавьте конфиг и создайте Pull Request.

  • Форкните репозиторий
  • Добавьте конфиг в configs.json
  • Создайте Pull Request
  • Ждите проверки и мержа
Форкнуть репозиторий

Требования к конфигам

Только для WoW 3.3.5

Конфиги должны быть совместимы с версией 3.3.5a

Популярные аддоны

ElvUI, WeakAuras, Details!, Plater, DBM/BigWigs

Без вредоносного кода

Конфиги проверяются на безопасность

Скриншоты приветствуются

Приложите скриншот интерфейса

Уникальный ID

Каждый конфиг должен иметь уникальный идентификатор
